Desafío N° 2 - JavaScript

Crear un algoritmo con un condicional

Código I

Este programa recibe como entradas los datos del usuario (nombre y apellido) y un string.

Luego, dependiendo de si la respuesta es afirmativa, negativa o inválida, le devuelve un mensaje al usuario.

Por último muestra en consola los datos del usuario y el estado.

Código II

En este código utilizo un ciclo WHILE para volver a preguntar cuando la respuesta es inválida.

Una vez que el usuario ingresa una respuesta válida, se le devuelve el mensaje correspondiente y se imprimen los datos en la consola.

También agregué la función toUpperCase() para optimizar la comparación de caracteres.
